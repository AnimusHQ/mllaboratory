# Отчёты DR/Backup Validation

Этот каталог содержит описание формата отчётов и шаблоны. Фактические отчёты **не** коммитятся по умолчанию.

## Формат dr-validate
Файл отчёта создаётся скриптом `closed/scripts/dr-validate.sh` и содержит:
- таймстемпы начала/окончания проверки,
- результаты readiness‑проверок,
- статус CRUD‑операций и upload/download,
- идентификаторы созданных объектов (без секретов),
- итоговый статус `pass|fail`.

Рекомендованный путь для отчёта — временный каталог (`/tmp`), либо корпоративное хранилище артефактов.
